## 🧭 About Me

I'm a **Senior Software Engineer** with **7+ years** of experience building and breaking large-scale distributed storage systems — and making sure they survive both.

Currently at **Oracle**, where I own test architecture and quality strategy for **Exadata** — designing scalable stress frameworks, driving test coverage from **50% → 80%**, and surfacing the kinds of distributed system failures that only show up under pressure.

Before Oracle, I spent 4 years at **Dell Technologies** automating infrastructure for **ObjectScale** (a Kubernetes-native object storage platform) — covering everything from UI to GraphQL APIs to underlying K8s microservices.

> Across my career, I've uncovered **400+ critical defects** — memory leaks, race conditions, security vulnerabilities, distributed system edge cases — and built the tooling to find them faster next time.
